Remember! Code snippets will use different IDs and classes to your code. You will need to name these suitably yourself in your code.

Try using Trinket yourself. This will enable you to check elements of your site in isolation to avoid or identify errors.

The code below has comments in it to explain key parts. These comments appear in green in the snippet. Comments are useful in your code also. Use them. Notice that HTML comments have a different syntax to CSS. If you add comments, it will be easier to remember what you code is aiming to achieve, and therefore identify problems later.

Notice how code is intended. This makes it easier to read the code. It is essential for you to indent code. It won't stop it working. It will make it easier to find problems.

When planning your responsive design, you may want to find your browser size - use http://howbigismybrowser.com/ to help you!
